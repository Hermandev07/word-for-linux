## Usage

To run the word application on your Linux system, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/Hermandev07/word-for-linux.git
```

2. Install dependencies:

```bash
cd word-for-linux
npm install
```

3. Start the application:

```bash
npm start
```
